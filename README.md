# Life Insurance Data Governance Portfolio

This portfolio demonstrates senior-level data governance design for a life insurance environment spanning LifePlus, FAST, SQL Server, PostgreSQL, Snowflake, AWS Redshift, Microsoft Purview, and Power BI.

## Professional Positioning
This portfolio is designed to reflect Senior / Lead Data Governance capability across governance operating models, data quality, metadata, data lineage, security, DLP, disaster recovery, and trusted reporting.

## Environment Modeled
- LifePlus
- FAST
- SQL Server
- PostgreSQL
- Snowflake
- AWS Redshift
- Microsoft Purview
- Power BI
- GitHub
- Azure DevOps (optional workflow layer)

## Governance Layers Demonstrated
- Data Governance and Stewardship
- Critical Data Elements and Business Glossary
- Data Quality Rules and Issue Management
- Metadata and Data Lineage
- Data Security and Data Loss Prevention (DLP)
- Disaster Recovery (DR) and Resilience Governance
- Reporting Certification and Trust

## Compliance / Privacy Overlays
These projects also include governance design considerations for:
- PCI DSS (where cardholder payment data is present)
- PHI / HIPAA (where protected health information applies)
- GDPR (where EU personal data is processed)
- CCPA (where California consumer personal information is in scope)

## Projects
1. Enterprise Policyholder & Producer Governance Framework
2. Data Quality Governance Workflow for Underwriting, Billing, and Claims
3. Metadata, Lineage, Security, and DR Trust Model
